# 朕即天下 · 皇帝模拟器 👑

> 一款决策驱动的中式宫廷网页小游戏。你是新登基的皇帝，群臣不断递上奏折，你的每一次抉择都牵动 **民心 · 国库 · 军事 · 朝堂** 四维国力——任何一维跌至谷底或盛极失衡，王朝都将倾覆。在位越久、四维越均衡，史书评价越高，争当 **千古一帝**！

🎮 **在线试玩**：开启 GitHub Pages 后即为 `https://<用户名>.github.io/<仓库名>/`

## 玩法
- 每张奏折由不同的大臣/角色呈上（丞相、大将军、皇后、御史、太监、番邦使臣……）
- 每个选项会改变四维数值，选项上的彩色小箭头**仅提示**受影响维度与升降方向
- 四维任一 **触底（≤0）** 或 **爆表（≥100）** 即触发对应亡国/驾崩结局
- 数十条剧情链与随机事件：权臣篡位、蛮族入侵、旱涝瘟疫、立储夺嫡、宦官专权、后宫争斗、变法改革、和亲外交、长生丹药、藩镇叛乱……
- 二十余种结局 + 庙号/谥号评定系统（千古一帝 ↔ 隋炀帝），结局图鉴收集

## 技术
- **纯 HTML / CSS / JavaScript，零框架、零依赖、零构建**，可直接静态托管
- 人物立绘、状态图标全部由 **程序化 SVG** 生成并内联（无外部图片）
- 音效用 **Web Audio API** 实时合成（无音频文件）
- localStorage 存档：结局图鉴 + 最长国祚记录 + 静音设置
- 移动端 / 桌面端自适应

## 文件结构
```
index.html        页面骨架（标题 / 游戏 / 结局 / 弹窗四屏）
css/style.css     中式宫廷主题样式（朱红 / 鎏金 / 宣纸）
js/audio.js       Web Audio 合成音效
js/assets.js      程序化 SVG 素材包（立绘 / 图标）
js/events.js      剧情数据库（事件 / 分支 / 结局）
js/game.js        游戏引擎（状态 / 调度 / 结算 / 评定）
```

## 本地运行
任意静态服务器即可，例如：
```bash
python -m http.server 8000
# 浏览器打开 http://localhost:8000
```
