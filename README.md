# 朕的江山 · 皇帝成长录 👑

> 一款多系统深度养成的**中式皇帝模拟经营**网页游戏，灵感源自《皇帝成长计划》。你从开国之君做起，培养帝王五维、统御满朝文武、经营后宫子嗣、统兵征战、理财科举外交，江山世代相传——力争青史留名为**千古一帝**，莫做亡国之君。

🎮 **在线试玩**：`https://plum-221.github.io/emperor-simulator/`

## ✨ 特色：满朝文武 · 全套 AI 赛璐璐立绘

游戏中所有立绘**全部由 ComfyUI（底模 animagineXL3.1）生成**，统一中国风赛璐璐古风、整盘 136 张：

- **皇帝**：少年 / 青年 / 中年 / 老年 四段年龄立绘，随帝王年岁自动更替（固定种子保持跨龄一致）
- **皇嗣**：婴孩通用图 + 襁褓 / 孩童 / 少年 / 弱冠 四段成长立绘
- **后宫嫔妃 24 · 文武百官 100 · 名将 4**：各具姿容，临幸 / 招贤 / 选秀时揭示

## 🏛️ 四大系统

| 系统 | 内容 |
|---|---|
| **朝堂** | 满朝文武各有文才/武略/忠诚/野心/性格；任命丞相、御史大夫、户部尚书、大将军、兵部尚书五大要职，他们每回合为你增益国力；可赏赐、处死。忠诚过低而野心过高者将**起兵谋反**。 |
| **后宫·皇嗣** | 临幸嫔妃→怀孕→生育皇子公主，培养并**立太子**。帝王驾崩由太子（或最年长皇子）继位，江山**世代相传**；绝嗣则亡国。 |
| **军务·战争** | 番邦入寇或主动北伐，遣大将军或御驾亲征；兵力＋将领武略＋陛下武力决定胜负，胜则开疆拓土、败则丧师失地，亲征更有阵亡之险。 |
| **经济·民生·科举·外交** | 国库靠民心疆域征税、养兵养官需开支；逢旱涝蝗疫赈灾，开海通商，三年一科举纳贤，行和亲、受朝贡。 |

## 🎲 玩法循环

回合制（一回合 = 一个月，十二月为一年）：
1. **上朝** —— 处理一桩朝政奏折（多选项，牵动国力与帝王属性）
2. **行动** —— 择一而行：勤政 / 读书 / 习武 / 养性 / 临幸后宫 / 召见群臣 / 休养
3. **下一回合** —— 结算经济、忠诚、生育、年岁

帝王五维（健康/智力/魅力/武力/政治）随行动成长；国库枯竭、民心尽失、兵败国破、权臣篡位、绝嗣，皆会**亡国**。在位长久、国力均衡、威望卓著者，方得**千古一帝**之名。

## 🛠️ 技术

- **纯 HTML / CSS / JavaScript，零框架、零构建**，可直接静态托管
- 立绘本地托管（不盗链），运行时读取 `assets/manifest.json` 动态分配角色
- 音效用 **Web Audio API** 实时合成；localStorage 自动存档、可续局
- 移动端 / 桌面端自适应

```
index.html         四屏骨架（标题 / 游戏 / 结局 / 面板）
css/style.css      中式宫廷主题样式
js/audio.js        Web Audio 合成音效
js/data.js         国家/帝王/官职/性格/事件库/角色构建
js/game.js         引擎：回合循环 / 经济结算 / 战争 / 传承 / 结局
js/ui.js           视图与交互：HUD / 帝王面板 / 事件 / 五大管理面板
assets/portraits/  真实古画立绘（公共版权）
assets/manifest.json  立绘清单
```
